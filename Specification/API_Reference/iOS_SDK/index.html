Device Connect SDK for iOS 詳解

Device Connect SDK for iOS は Device Connect API に対応したiOS向けのデバイスプラグイン及び、UIアプリを開発するためのSDKである。
本SDKでは以下の機能を提供する。

  • Device Connect Managerの設定、起動
  • Device Connect APIへのアクセス機能
  • デバイスプラグイン開発用クラスの提供
  • URI生成機能
  • イベント操作機能
  • メッセージ操作機能
