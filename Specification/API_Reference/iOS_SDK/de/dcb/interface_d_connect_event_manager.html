詳解

イベント管理機能を提供する。

デバイスプラグインに対してイベントの管理機能を提供する。 イベントデータの追加、削除、検索の機能を提供する。 イベントデータの操作処理は DConnectEventCacheController の実態に依存する。 例として、イベントデータをDBに保存し永続化したい場合は DConnectDBCacheController を 設定する。
