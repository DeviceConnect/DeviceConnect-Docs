接続されているデバイス一覧の取得リクエストを受け取ったことをデリゲートに通知する。

profileが接続されているデバイス一覧の取得リクエストを受け取ったことをデリゲートに通知する。
実装されない場合には、Not supportのエラーが返却される。

引数

NOレスポンスパラメータを返却しないので、DConnectManager::sendResponse:で返却すること。

onservicechangeイベント登録リクエストを受け取ったことをデリゲートに通知する。

実装されない場合には、Not supportのエラーが返却される。

ServiceDiscovery プロファイル。

Service Discovery Profileの各APIへのリクエストを受信する。 受信したリクエストは各API毎にデリゲートに通知される。
